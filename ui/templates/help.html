How to Play

Hollow Host is an AI-powered text adventure game. You interact with the game by typing commands or actions, and the AI Dungeon Master responds with descriptions of what happens.

You can type natural language commands like "I search the room" or "I talk to the innkeeper", or use specific commands like "/roll" to perform dice rolls.

Dice Rolls

To roll dice, use the /roll command followed by the dice notation. For example:

  • /roll 1d20 - Roll a 20-sided die
  • /roll 2d6 - Roll two 6-sided dice
  • /roll 1d20+STR - Roll a 20-sided die and add your Strength modifier

Character Stats

Your character has six primary stats that affect various actions:

  • STR (Strength): Physical power, melee attacks, lifting
  • DEX (Dexterity): Agility, reflexes, stealth, ranged attacks
  • CON (Constitution): Health, stamina, resistance to poison
  • INT (Intelligence): Knowledge, memory, arcane magic
  • WIS (Wisdom): Perception, intuition, divine magic
  • CHA (Charisma): Personality, leadership, persuasion

About Hollow Host

Hollow Host is a modular solo/multiplayer text-based RPG system powered by a local LLM via LM Studio. The AI serves as the Dungeon Master (DM), enabling immersive tabletop-style RPGs through natural language, character systems, and rule-based gameplay.
